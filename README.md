<p><strong>　　《魔法气泡eSports》收录了经典《魔法气泡通》以及可以轻松游玩大连锁的《魔法气泡狂热》2种游玩规则。除了大家所熟悉的角色外，也从《魔法气泡Quest》、《魔法气泡编年史》等作品里选出总共24名角色登场。另外还搭载了连线对战与离线大会模式等。</strong></p>
